planning_client Module

This ROS node is used for sending and cancelling the goal using the action server for, PlanningAction count the number of reached and cancelled goals through the /counter service and for publish informations, received by the /odom topic, to the /posvel topic.

Subscribes to:

/odom

Publishes to:

/posvel

Clients:

/counter

Action:

/reaching_goal/goal /reaching_goal/cancel

information Module

This ROS node is used for printing the current goal distance and the average of x and z velocity. The current goal is obtained from the /reaching_goal/goal topic , then the current goal distance is computed by the difference between the current goal and the current position, received from /posvel topic. The current x and z velocities are obtained from /posvel topic, then an average of these velocities is done.

Subscribes to:

/reaching_goal/goal /posvel
